# MovieDB (IMDB clone)

+ Authentication (Register, Login)
+ Profile (Avatar, Bio, Edit Email/Username, Total Watch Time, Seen Movie List, Watch Later List )
+ Movies (Search, Movie Details, Popular/Upcoming/Top Rated Lists)
